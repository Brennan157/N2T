# N2T

A React Native app built with Expo (SDK 55).  
N2T is a meal‑planning app focused on saving money and reducing food waste.

## Getting Started

Install dependencies:

npm install

Run the project with the dev client:

npx expo start --dev-client

## Development Build

To generate a new dev build:

eas build --profile development --platform android
eas build --profile development --platform ios

## Tech Stack

- Expo SDK 55
- React Native 0.83
- React 19
- EAS Build (for custom dev clients)
